# (自分・初心者用）コミット等の仕方のメモ
現在の状態を確認...`git status`
ファイルをインデックスに追加...`git add <FILE>`
インデックス内のファイルをコミット...`git commit -m "<MESSAGE>"`
{初期}リモートレポジトリを指定...`git remote add origin https://github.com//awesome.git`
ローカルレポジトリをプッシュする...`git push origin <BRANCH_NAME>`
